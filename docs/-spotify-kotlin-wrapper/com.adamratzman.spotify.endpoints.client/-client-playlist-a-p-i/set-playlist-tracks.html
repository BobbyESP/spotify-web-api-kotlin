setPlaylistTracks

fun setPlaylistTracks(playlist: String, vararg tracks: String): SpotifyRestAction<Unit>

Replace all the tracks in a playlist, overwriting its existing tracks. This powerful request can be useful for replacing tracks, re-ordering existing tracks, or clearing the playlist.

Parameters

playlist - the spotify id or uri for the playlist.

tracks - The Spotify track ids.

Exceptions

BadRequestException - if playlist is not found or illegal tracks are provided
