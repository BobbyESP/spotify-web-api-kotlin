getUserProfile

fun getUserProfile(): SpotifyRestAction<SpotifyUserInformation>

Get detailed profile information about the current user (including the current user’s username).

The access token must have been issued on behalf of the current user. Reading the user’s email address requires the user-read-email scope; reading country and product subscription level requires the user-read-private scope. Reading the user’s birthdate requires the user-read-birthdate scope.

Return
Never-null SpotifyUserInformation object with possibly-null country, email, subscription and birthday fields
